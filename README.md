# IA-III

Trabalho de aprendizado supervisionado com dados abertos do SUS (SINASC).

## Setup

```bash
uv sync
```

## Uso

```bash
uv run jupyter lab notebooks/
```

## Estrutura

- `ml/` — modelos e pipeline
- `notebooks/` — análises
- `scripts/` — ETL e utilitários
- `data/` — dados locais (gitignored)
- `slides/` — apresentação
